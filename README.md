## 功能
  - 日期資訊（日期、可購買民眾之身分證末碼等）
  - 縣市、地區、選單搜尋
  - 地址關鍵字搜尋
  - 點擊圖標時跳出該藥局資訊
  - 自動判斷數量，顯示於地圖 icon
    - 成人、兒童都有貨，顯示藍色
    - 成人、兒童都沒貨，顯示灰色
    - 成人、兒童單一有貨，顯示紅色
